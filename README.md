# 🔥 Streak Tracker Pro

A responsive streak and habit tracking web application built using HTML, CSS, and JavaScript.

## 🚀 Features
- Daily streak tracking
- Target & unlimited goals
- Progress visualization
- Categories & priority system
- Notes per task
- Dark/Light mode
- Local storage persistence
- Export / Import backup

## 🛠 Tech Stack
- HTML
- CSS
- JavaScript

## 📌 Purpose
Helps users build consistency and track daily habits effectively.
